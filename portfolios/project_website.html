Built this Portfolio Website from Scratch

I started with a basic HTML website which was pretty barebones as you can see below. I didn't know anything about front-end development so I spent a couple days watching youtube tutorials and courses in order to teach myself CSS and some JS.

After learning the basics of CSS I styled up the site and blocked the text and images to make it look professional. Once the main functionalities of the website were done, I created a repo for it on GitHub. Then, I just used GitHub Pages to host the site.

In the future I might add some animations to make it look fancier.
